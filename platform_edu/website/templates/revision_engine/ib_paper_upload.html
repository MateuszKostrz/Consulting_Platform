Paper folder → HTML viewer

Put your export in a folder next to an images subdirectory (as in your Digitary / LaTeX export), containing a .tex file and referenced figures. Then either zip that folder, or use the folder picker (Chrome / Edge) to upload the tree in one go (80 MB max).
